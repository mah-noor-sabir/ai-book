# روبوٹک نروس سسٹم کا آرکیٹیکچر

یہ دستاویز روبوٹک نروس سسٹم کے مختلف آرکیٹیکچرل پیٹرنز کی وضاحت کرتی ہے۔

## مدولی ڈھانچہ

روبوٹک نروس سسٹم کو مختلف ماڈیولز میں تقسیم کیا گیا ہے:

- سینسر ماڈیول
- کنٹرول ماڈیول
- ایکچو ایٹر ماڈیول
- کمیونیکیشن ماڈیول

## کمیونیکیشن کے طریقے

- پبلشر/سبسکرائیبر
- سروس کالز
- ایکشن سرورز
