# ضروریات اور ترتیب کا گائیڈ

## جائزہ

گزیبو اور یونٹی کے ساتھ ڈیجیٹل ٹوئن کے نفاذ میں گہرائی سے جانے سے پہلے، آپ کو اپنے ترقی کے ماحول کو ضروری سافٹ ویئر اور علم کے ساتھ ترتیب دینے کی ضرورت ہے۔ یہ گائیڈ آپ کو اس کورس کے تمام ماڈیول کو کامیابی سے مکمل کرنے کے لیے ضروری تمام ضروریات سے گزرنے لے جائے گا۔

## تکنیکی تقاضے

### آپریٹنگ سسٹم
- **تجویز کردہ**: Ubuntu 22.04 LTS
- **متبادل**: Windows 10/11 WSL2 کے ساتھ (Windows Subsystem for Linux)
- **MacOS**: تعاون یافتہ لیکن اضافی ترتیب کی ضرورت ہو سکتی ہے

### ہارڈ ویئر تقاضے
- **CPU**: Multi-core پروسیسر (Intel i5 یا اس کے مساوی)
- **RAM**: 8GB کم از کم، 16GB تجویز کردہ
- **GPU**: OpenGL 3.3+ سپورٹ کے ساتھ مخصوص گرافکس کارڈ (یونٹی کے لیے ضروری)
- **اسٹوریج**: مکمل تنصیب کے لیے 10GB خالی جگہ
- **نیٹ ورک**: ڈاؤن لوڈز اور اپ ڈیٹس کے لیے مستحکم انٹرنیٹ کنکشن

## سافٹ ویئر تنصیب

### ROS 2 تنصیب (Humble Hawksbill)

یوبنٹو صارفین کے لیے:

```bash
# لوکل سیٹ اپ
sudo locale-gen en_US en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8
export LANG=en_US.UTF-8

# ذرائع سیٹ اپ کریں
sudo apt update && sudo apt install -y curl gnupg lsb-release
curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key | sudo gpg --dearmor -o /usr/share/keyrings/ros-archive-keyring.gpg

echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

# ROS 2 انسٹال کریں
sudo apt update
sudo apt install ros-humble-desktop
sudo apt install ros-humble-gazebo-ros-pkgs
sudo apt install python3-colcon-common-extensions

# ROS 2 سرچشمہ
source /opt/ros/humble/setup.bash
```

### گزیبو تنصیب

گزیبو گارڈن انسٹال کریں:

```bash
curl -sSL http://get.gazebosim.org | sh

# تنصیب کی تصدیق کریں
gz --version
```

### یونٹی تنصیب

1. یونٹی ہب کو https://unity.com/download سے ڈاؤن لوڈ کریں
2. یونٹی ہب انسٹال کریں اور اکاؤنٹ بنائیں
3. یونٹی ہب کے ذریعے یونٹی 2021.3 LTS انسٹال کریں
4. لینکس ڈویلپمنٹ کے لیے اضافی ماڈیولز انسٹال کریں اگر ضرورت ہو

## علمی ضروریات

### اہم علمی علاقے

اس کورس میں شروع کرنے سے پہلے، آپ کو یہ ہونا چاہیے:

#### ROS کی بنیادیں
- ROS 2 تصورات کا علم (Nodes, Topics, Services, Actions)
- ROS 2 ٹولز کا تجربہ (ros2 run, ros2 launch, وغیرہ)
- ROS 2 ورک سپیس تخلیق اور کمپائلیشن کا علم

#### پائی تھون پروگرامنگ
- بنیادی سے درمیانی پائی تھون کا علم
- آبجیکٹ اورینٹڈ پروگرامنگ تصورات کا علم
- روبوٹکس کے لیے پائی تھون لائبریریز کا تجربہ

#### لینکس کمانڈ لائن
- bash کمانڈز اور فائل مینجمنٹ کے ساتھ آرام دہ
- پیکج مینجمنٹ (apt/yum) کا علم
- ٹیکسٹ ایڈیٹرز (nano، vim، یا اسی طرح) کا تجربہ

#### 3D تصورات
- کوآرڈینیٹ سسٹم کا بنیادی علم (ورلڈ، مقامی، کیمرہ)
- ٹرانسفارمیشنز (rotation، translation، scaling) کا علم
- بنیادی جیومیٹرک تصورات کا علم

#### فزکس کی بنیادیں
- کنیمیٹکس کا علم (position، velocity، acceleration)
- ڈائنیمکس کا علم (forces، torques، mass)
- فرکشن اور کالیژن تصورات کا بنیادی علم

## تجویز کردہ سیکھنے کے وسائل

### ROS 2 وسائل
- [ROS 2 Humble دستاویزات](https://docs.ros.org/en/humble/)
- ROS 2 ٹیوٹوریلز اور مثالیں
- "Programming Robots with ROS" by Morgan Quigley

### پائی تھون وسائل
- پائی تھون سرکاری دستاویزات
- NumPy اور SciPy ٹیوٹوریلز سائنسی کمپیوٹنگ کے لیے
- آبجیکٹ اورینٹڈ پروگرامنگ تصورات

### 3D اور گرافکس
- بنیادی 3D ریاضی کے تصورات
- کوآرڈینیٹ سسٹم ٹرانسفارمیشنز
- OpenGL یا گرافکس پروگرامنگ کی بنیاد

## ترتیب کی تصدیق

### ROS 2 تنصیب کی تصدیق کریں

```bash
# ROS 2 تنصیب چیک کریں
source /opt/ros/humble/setup.bash
ros2 topic list

# بنیادی فعالیت کا ٹیسٹ کریں
ros2 run demo_nodes_cpp talker &
ros2 run demo_nodes_cpp listener
```

### گزیبو تنصیب کی تصدیق کریں

```bash
# گزیبو لانچ کریں
gz sim
# ڈیفالٹ ماحول کے ساتھ گزیبو GUI کھولنا چاہیے
```

### یونٹی تنصیب کی تصدیق کریں

1. یونٹی ہب لانچ کریں
2. ایک نیا 3D پروجیکٹ بنائیں
3. تصدیق کریں کہ ایڈیٹر بغیر کسی خامی کے کھلتا ہے

## عام مسائل کا حل

### ROS 2 مسائل
- **مسئلہ**: ROS 2 کمانڈز نہیں مل رہے
  - **حل**: یقینی بنائیں کہ ROS 2 ماحول سورسڈ ہے: `source /opt/ros/humble/setup.bash`

- **مسئلہ**: گزیبو کے ساتھ اجازت کی خامیاں
  - **حل**: صارف کی اجازتیں اور گرافکس ڈرائیورز چیک کریں

### یونٹی مسائل
- **مسئلہ**: یونٹی سٹارٹ اپ پر کریش ہو جاتا ہے
  - **حل**: گرافکس ڈرائیورز اپ ڈیٹ کریں، `-force-opengl` فلیگ کے ساتھ چلانے کی کوشش کریں

- **مسئلہ**: یونٹی میں کارکردگی کے مسائل
  - **حل**: ہارڈ ویئر تقاضے چیک کریں، رینڈرنگ کی کوالیٹی کم کریں

### نیٹ ورک مسائل (یونٹی-ROS کنکشن)
- **مسئلہ**: یونٹی کو ROS سے مربوط نہیں کر سکتے
  - **حل**: فائر وال سیٹنگز چیک کریں، IP ایڈریس کی تصدیق کریں، یقینی بنائیں کہ ROS برج چل رہا ہے

## اختیاری اضافے

### اعلی درجے کے ٹول (اختیاری)
- **RViz2**: ROS ویژولائزیشن کے لیے
- **Gazebo Web Interface**: براؤزر بیسڈ سیمولیشن مانیٹرنگ کے لیے
- **Unity ML-Agents**: روبوٹکس کے ساتھ مضبوط کریکٹر لرننگ کے لیے
- **Docker**: کنٹینرائزڈ ڈویلپمنٹ ماحول کے لیے

### اضافی سینسرز (اختیاری)
- فورس/ٹورک سینسرز
- GPS سیمولیشن
- تھرمل کیمرے
- سٹیریو وژن کے لیے ملٹی-کیمرے

## شروع کرنے کی چیک لسٹ

کورس میں آگے بڑھنے سے پہلے، یقینی بنائیں کہ آپ نے یہ مکمل کر لیا ہے:

- [ ] ROS 2 Humble Hawksbill انسٹال کیا
- [ ] گزیبو گارڈن انسٹال کیا
- [ ] یونٹی 2021.3 LTS انسٹال کیا
- [ ] تمام تنصیبات کام کر رہی ہیں کی تصدیق کی
- [ ] یونٹی روبوٹکس پیکیج انسٹال کیا
- [ ] ROS-TCP-کنیکٹر انسٹال کیا
- [ ] بنیادی ROS فعالیت کا ٹیسٹ کیا
- [ ] گزیبو سیمولیشن چلایا
- [ ] ایک بنیادی یونٹی منظر بنایا
- [ ] یونٹی کو ROS سے مربوط کیا (اعلی درجے کے صارفین کے لیے اختیاری)

## اگلے اقدامات

ایک بار جب آپ تمام ضروریات مکمل کر لیتے ہیں:

1. ڈیجیٹل ٹوئنز کے تعارف کے باب سے شروع کریں
2. ہر ماڈیول کو ترتیب سے آگے بڑھیں
3. ہر باب میں ہاتھوں سے کام کرنے والی مشقیں مکمل کریں
4. حتمی ماڈیول میں اپنا خود کا ڈیجیٹل ٹوئن سسٹم تیار کریں

یاد رکھیں کہ اس کورس میں کامیابی کے لیے ترقی کا ماحول مناسب طریقے سے ترتیب دینا ناگزیر ہے۔ ہر جزو کے صحیح کام کرنے کی تصدیق کے لیے وقت لگائیں قبل از اس کے آگے بڑھیں۔
