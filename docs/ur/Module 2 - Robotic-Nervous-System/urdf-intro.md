# URDF کا تعارف

URDF (Unified Robot Description Format) روبوٹ کی جسمانی ساخت کی تفصیل کے لیے استعمال ہوتا ہے۔

## URDF کی تعریف

URDF ایک XML فارمیٹ ہے جو روبوٹ کی جسمانی خصوصیات کی تفصیل فراہم کرتا ہے:

- جوڑ (joints)
- لنکس (links)
- سینسرز
- کنٹرولرز
