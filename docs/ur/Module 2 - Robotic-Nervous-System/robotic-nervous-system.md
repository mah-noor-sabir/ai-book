# روبوٹک نروس سسٹم

روبوٹک نروس سسٹم روبوٹ کے لیے ایک کنٹرول اور کمیونیکیشن ڈھانچہ فراہم کرتا ہے۔

## کمیونیکیشن پروٹوکولز

- ROS (Robot Operating System)
- DDS (Data Distribution Service)
- CAN بس

## کنٹرول آرکیٹیکچر

روبوٹک نروس سسٹم کے ڈھانچے میں درج ذیل شامل ہیں:

- سینسر ان پٹس
- کنٹرول الگورتھم
- ایکچو ایٹر آؤٹ پٹس
