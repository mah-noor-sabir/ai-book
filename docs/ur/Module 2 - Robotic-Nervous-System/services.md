# سروسز

یہ صفحہ روبوٹک نروس سسٹم کی سروسز کے بارے میں ہے۔

## سروسز کی اقسام

روبوٹک نروس سسٹم میں مختلف اقسام کی سروسز ہوتی ہیں:

- کمیونیکیشن سروسز
- کنٹرول سروسز
- سینسر سروسز

## سروسز کا استعمال

یہاں سروسز کے صحیح استعمال کے بارے میں معلومات ہوں گی۔
