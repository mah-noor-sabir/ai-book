# ٹاپکس

یہ صفحہ روبوٹک نروس سسٹم کے ٹاپکس کے بارے میں ہے۔

## ٹاپکس کی اقسام

روبوٹک نروس سسٹم میں مختلف اقسام کے ٹاپکس ہوتے ہیں:

- سینسر ڈیٹا ٹاپکس
- کنٹرول کمانڈ ٹاپکس
- اسٹیٹس ٹاپکس

## ٹاپکس کا انتظام

یہاں ٹاپکس کے انتظام کے بارے میں معلومات ہوں گی۔
